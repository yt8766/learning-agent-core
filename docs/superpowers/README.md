状态：current

本目录存放功能演进的实施计划（plan）与设计规格（spec）。

- `plans/` — 带 checkbox 的实施计划，按日期 + 主题命名
- `specs/` — 对应计划的技术设计文档

与 `docs/workflows/` 的区别：`workflows/` 记录已稳定的工作流程与执行流转；`superpowers/` 记录正在推进或已完成的功能演进计划与设计。
