# Learning Flow Checklist

## 类型

- `TaskRecord` 是否仍有：
  - `externalSources`
  - `reusedSkills`
  - `learningEvaluation`
- `ChatCheckpointRecord` 是否仍有：
  - `externalSources`
  - `reusedSkills`
  - `learningEvaluation`

## 后端

- LearningFlow 是否仍能生成候选
- checkpoint 是否仍能投影学习状态
- evidence/source 是否仍能和 learning 关联

## 前端

- `agent-chat` 是否仍显示 learning suggestions
- `agent-admin` 是否仍能显示 Learning Center
- `agent-admin` 是否仍能显示 Skill Lab 生命周期信息

## 方向性检查

- 是否仍朝“主动研究 -> 评估 -> 沉淀 -> 复用”推进
- 是否退回成“只展示候选，不参与执行闭环”
- 是否有高置信自动沉淀方向
- 是否支持后续技能复用与治理
